# python-daily-coding-tasks

**🧩 Problem Statement**

Write a Python program that takes a list of integers from the user and counts how many numbers are:

Positive

Negative

Zero

This type of problem is commonly asked in interviews and is useful in real-world cases such as analysing sensor values or signal data.

**🛠️ Concepts Used**

Input handling

String to integer conversion

Lists

For loop

Conditional statements (if, elif, else)

**🧠 Logic**

Take numbers from the user separated by spaces.

Split the input into a list.

Convert each value into an integer.

Maintain three counters:

positive

negative

zero

Traverse the list and update the counters.

Print the final counts.


▶️ Sample Input
5 -8 6 9 0 5 4 0 -3 5 7
▶️ Sample Output
Positive Numbers = 7
Negative Numbers = 2
Zeros = 2
